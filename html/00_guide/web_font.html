🍙 Web Font?

웹사이트에서 사용하는 폰트가 시스템에 없으면, 서버에서 불러와 웹페이지에서 원하는 폰트를 사용 할 수 있게 해준다.
예쁜 폰트를 노출할수있지만, 파일이 무거워서 그대로 사용하면 FOIT, FOUT 현상이 발생 할 수 있다.

1) FOIT (Flash of Invisible Text)

웹폰트 로딩이 되기 전까지 아무것도 안 뜨는 현상이다. 3초 동안 웹폰트가 로드 되기를 기다린다. 3초가 지나도 로딩이 안되면 기본폰트를 보여준다.
크롬, 파이어폭스, 사파리의 기본동작

2) FOUT (Flash of Unstyled Text)

웹폰트가 로드될때까지 시스템의 기본 폰트를 보여주고 로드가 되면 글꼴을 대체하는 방식
IE와 EDGE는 FOUT이다

3) 특이사항

웹폰트는 같은 해상도라도 윈도우에서 더 안좋게 표현된다.
한글에서 word-break: keep-all 자동으로 적용 안되는 경우가 있다.
한국어는 기본적으로 break-all이 작동했다 생각해서 간혹 한글자만 내려가는 경우가 있다. 이 경우 css로 적용 해야 한다.

🍙 웹폰트 확장자

웹사이트에서 사용하는 폰트가 시스템에 없으면, 서버에서 불러와 웹페이지에서 원하는 폰트를 사용 할 수 있게 해준다.
예쁜 폰트를 노출할수있지만, 파일이 무거워서 그대로 사용하면 FOIT, FOUT 현상이 발생 할 수 있다.

1) EOT

마이크로 소프트가 예전에 만든 것으로 font-face를 처음 도입했다.
Ie8이하 버전은 @font-face를 사용할 때 .eot파일만 인식 참고) 곰랩 개편할때 ie8에 적용했는데, 작동이 안해서 적용 못함

2) OTF (open type)

OTF, TTF 형식은 쉽게 복제되서 대신 WOFF형식이 만들어짐
지원 브라우저 - Chrome 4+, Firefox 3.5, Opera 10+, Safari 3-5, Android, iOS 에서 지원

3) WOFF

모질라와 다른 조직들이 결합하여 웹에서 사용할 목적으로 개발했다.
OTF/TTF 를 이용한 구조의 압축된 버전으로, 다른 글꼴 형식보다 빠르게 로드된다.
이 형식은 글꼴 파일 안에, metadata나 license 정보도 포함할 수 있다.
Chrome 6+, Firefox 3.6+, IE 9+, Safari 5.1+, FF Mobile(Gecko) 5, Opera Mobile11 에서 지원

4) WOFF2

다른 포맷보다 파일사이즈가 30% 적다.
Chrome 6+, Firefox 3.6+, IE 9+, Safari 5.1+, FF Mobile(Gecko) 5, Opera Mobile11 에서 지원

5) SVG, SVGZ** (svg의 압축버전)

파일 크기를 가볍게 하여, 모바일 사용에 적합하다.
Chrome, Safari, Safari Mobile, Opera Mobile10 에서 지원, IE, Fire fox 지원 x

🍙 웹폰트 경량화

💡 경량화를 해야 하는 이유

웹폰트를 다운받는 시간 동안 텍스트가 노출이 안되고, 용량이 커서 퍼포먼스가 떨어진다. 특히 한글, 한자의 경우 더 용량이 크다.
오리지널 폰트 굵기 개당 용량이 `16MB` 만약 사이트에서 모든 굵기를 불러오면 폰트 불러오는데만 약 150MB
경량화를 하게 되면 사용할 굵기 별로 woff, woff2, eot을 사용하면 되는데, 개당 약 300kb로 용량이 확줄어든다.
단점 - 서브셋에 넣지 않은 글자는 해당 폰트가 적용되지 않아 시스템 폰트로 보인다.
여기에서 경량화한 폰트에 포함 되는 서브셋 (한글, 영어, 일본어, 일본어 한자)

💡 경량화 방법

  • 0 폰트 메이커, Woff 컨버터 설치 후 폰트 메이커 실행
    자동 실행 되기 때문에 2개다 설치 후 진행
  • 1-1 컴퓨터에 설치된 폰트 불러오기
  • 1-2 폰트 파일 불러오기
  • 2 새로운 폰트를 저장할 경로지정
  • 3-1 서브셋 파일 불러오기(txt)
  • 3-2 서브셋 붙여 넣기
    3-1, 3-2 둘중 마음에 드는 방법으로 사용하면 된다
  • 4 파일명이 아닌 폰트 패밀리의 이름을 지정하는 것이다. 체크 안함
  • 5 체크하면 폰트 파일이 생성된 후 WOFF 컨버터가 자동으로 실행되게 한다.
  • 6 완료버튼을 누르면 변환이 시작된다
  • 0 경량화 완료후 WOFF 컨버터 실행
  • 1 WOFF 컨버터가 자동으로 실행되게 했기 때문에 입력되어 있다.
    입력이 안되있음 파일 추가하면 됨
  • 2 새로운 폰트를 저장할 경로를 지정한다
  • 3 같이 변환 할 확장자 선택 Woff, woff2, eot 등이 있다
  • 4 변환완료
    otf파일로 하면 eot파일 변환은 안되서 변환되는 웹사이트에서 변환함

🍙 웹폰트 적용 방법


/*
	브라우저에 따라서 나열된 폰트 형식 중 필요한 하나의 폰트를 다운받아 받아와 사용한다.
	적용되기를 원하는 순서대로 폰트 포맷을 나열
	가벼운 순으로 정의하는게 좋다.
	ex) 순서대로 불러 오기 때문에,
	    무거운 eto 폰트가 지원되는 브라우저면 가벼운 폰트 대신 무거운 폰트를 쓰게 된다
	    미디어 쿼리 안에 font-face을 선언하면 특정 해상도에서만 불러올수있다
*/

@font-face {
  font-family: "NotoSans";
  font-weight: 100;
  src:local("※"), url(".../NotoSansCJK_Light.svg") format("svg")
	    local("※"), url(".../NotoSansCJK_Light.woff2") format("woff2"),
			local("※"), url(".../NotoSansCJK_Light.woff") format("woff");
}
@font-face {
  font-family: "NotoSans";
  font-weight: 400;
	src:local("※"), url(".../NotoSansCJK_Medium.svg") format("svg")
	    local("※"), url(".../NotoSansCJK_Medium.woff2") format("woff2"),
			local("※"), url(".../NotoSansCJK_Medium.woff") format("woff");
}

body{
	font-family: "NotoSans";
	font-weight:100; //NotoSansCJK_Light 적용
}
strong{
	font-weight:400;//NotoSansCJK_Medium 적용
}

💡 다른 적용 방법

  • 1) google에서 지원하는 @import, js로딩, link 링크 적용
    간편하고, 최적화도 되어있어 많이 사용한다.
    구글에서 불러 오는거라 CDN 서버에 문제가 생겼을 때 사이트에 적용이 안되고 에러가 발생한다.
    이전에 이런 경우가 생겨 웹이 깨진 경우가 있어서 js, css 등 사용하는 모든 파일은 회사 서버에 올려 사용 하기를 권장
  • 2) base64로 인코딩
    성능 최적화에 좋다고해서 적용 해봤는데, 용량이 늘어나고 아니라서 처음 불러올때는 로딩 속도가 느리고 용량이 생각 크다.
    반응형에 적용하기 무리가 있어서 사용 안 함.

💡 웹폰트 관련 스타일

  • - FONT-WEIGHT

    @font-face에서 font-weight 특정 값에 원하는 굵기의 파일을 넣으면, font-weight로 원하는 굵기를 불러 올수있다.

  • - FONT-DISPLAY

    브라우저가 폰트를 어떻게 로딩 할 것 인지 설정
    auto : 브라우저의 기본동작에 맡기는 방식이다.
    block : 타임아웃까지 텍스트를 보여주지 않는다. (FOIT)
    swap : 응답이 올때까지 무한정 기다리고 그 전 까진 바로 기본폰트를 보여준다. 반드시 적용해야 하는 폰트일 경우에 쓸 수 있다.
    fallback : 최대 100ms 동안 block을 하고 기본폰트를 보여준다. 응답이 오면 해당 폰트로 변경 된다.
    optional : 최대 100ms 동안 block을 하고 기본폰트를 보여준다. 그 후에 응답이 되도 대체를 하지 않는다.

  • - LOCAL()

    font-face을 사용하면, 시스템이나 쿠키에 폰트가 있어도 서버에서 불러온다.
    local()을 선언하면, 설치되어있는 폰트나 쿠키에 있는 폰트를 사용한다.

  • - Unicode-Range

    
/*
	해당 폰트가 지원할 범위를 설정 할 수 있다.
	사이트에 해당 범위의 문자가 있을 때에만 폰트를 다운받도록 변경이 가능하다.
	모든 모던 브라우저 지원
*/

@font-face {
	font-family: "NotoSans";
	src: local("※"), url(".../NotoSansCJK_Light.woff") format("woff");
	unicode-range: U+000-5FF;
/*
	단일 코드 포인트(예: U+416)
	간격 범위(예: U+400-4ff): 범위의 시작 및 끝 코드 포인트를 나타냅니다.
	와일드 카드 범위(예: U+4??): '?' 문자는 임의의 16진수를 나타냅니다.
*/
}
