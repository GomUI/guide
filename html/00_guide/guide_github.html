🥨 업데이트시 주의사항

★ Github(https://github.com/GomUI/guide) ★

★ 만약 계정 정보등이 포함된 링크의 경우에는 class="no_open"을 붙여서 업데이트 ★

🥨 UI 가이드 Github 업데이트

🍩 Github 사용

1) Github 접속 후 UI팀 구글 계정(teamUIdev@gmail.com / ui11ui!@)으로 로그인

2) 수정하고자하는 파일로 접근해 우측 Edit 버튼 클릭

3) 파일을 모두 수정하고나서 하단의 커밋메세지 작성 후 Commit changes 버튼 클릭


🍩 Git 사용

1) Github 프로젝트를 clone 받고 변경된 가이드 html만 업데이트

2) 터미널에서 git add, commit 후 push 시 아래와 같이 GitHub Login 팝업이 뜨면 UI팀 구글 계정(teamUIdev@gmail.com / ui11ui!@)으로 로그인

3) 로그인 후 유저 네임 입력(GomUI)

4) 팝업이 뜨면 패스워드 입력(ui11ui!@)

5) 모두 입력하면 아래와 같이 업로드 완료
