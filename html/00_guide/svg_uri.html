🐳 SVG URI 활용하기

Sass를 통해 색상 변환하기

Data URI를 함수화하여 색상 변경이 가능하여 마우스오버 등 색상만 변경하는 경우 함수로 대응
1) 아래와 같이 함수를 하나 만들어줌 (SVG URI 코드에서 색상 지정하는 부분을 변수화) 

@function icoInfo($color){
	$url:"data...fill:%23#{$color};...";
	@return url($url);
}

2) 함수 사용하여 style 작성 

.ico_info{
	background-image:icoInfo('ff4352');
	&:hover{
		background-image:icoInfo('f9acb2');
	}
	&.black{
		background-image:icoInfo('333') !important;
	}
	&.blue{
		background-image:icoInfo('10adff') !important;
	}
}

🐳 SVG URI 함수의 매개변수 늘려서 사용하기

색상 외에 추가적으로 변수처리 할 수 있는 부분 변수처리하기 

@function arrow($dir, $color){
	$url: '';
	@if $dir == up{
		$url:"data...fill:%23#{$color};...";
	}
	@if $dir == down{
		$url:"data...fill:%23#{$color};...";
	}
	@if $dir == right{
		$url:"data...fill:%23#{$color};...";
	}
	@if $dir == left{
		$url:"data...fill:%23#{$color};...";
	}
	@return url($url);
}

@function btnClose($color,$opacity){
	$url:"data...fill:%23#{$color};opacity:#{$opacity}...";
	@return url($url);
}

.ico_arrow{
	...
	&.up{
		background-image:arrow('up', '333');
	}
	&.down{
		background-image:arrow('down', '333');
	}
	&.left{
		background-image:arrow('left', '333');
	}
	&.right{
		background-image:arrow('right', '333');
	}
	&.up_red{
		background-image:arrow('up', 'ff4352');
	}
	&.down_red{

	}
	&.left_red{

🐳 SVG URI 사용시 주의 할 점

  • 1) Data URI값에는 hex 코드를 넣을 수 없다.
  • 2) SVG 태그로 filter가 들어 간 경우, " <filter>"의경우 ie에서 그림자 표현이 안됨
  • 3) SVG 코드상 width나 height가 다르다면, 함수 사용 불가
